# MVP 版本规划

## MVP 定义

Minimum Viable Product - 最小可行产品，提供核心功能，验证产品价值。

---

## MVP 功能范围

### ✅ 包含的功能

1. **文本联想**
   - 文本输入和分析
   - 情绪/关系/场景识别
   - 热梗和文化内容联想
   - 匹配度评分

2. **热梗库**
   - 基础热梗数据（100+）
   - 热梗列表浏览
   - 热梗搜索

3. **文化联想库**
   - 基础文化内容（50+）
   - 分类浏览
   - 内容详情

4. **基础文案生成**
   - 抖音文案
   - B站文案

5. **基础 UI**
   - 首页
   - 文本联想页
   - 热梗库页
   - 文化库页

### ❌ MVP 暂不包含

- 图片联想
- 视频联想
- 素材推荐
- 用户系统
- 历史记录
- 小红书和公众号文案
- 高级筛选

## MVP 技术实现

### 后端技术栈

- FastAPI
- PostgreSQL 16
- Qdrant
- OpenAI API
- Sentence-Transformers (all-mpnet-base-v2)

### 前端技术栈

- Next.js 14
- TypeScript
- TailwindCSS
- Shadcn UI

### AI 模型使用

| 功能 | 模型 |
|------|------|
| 文本分析 | OpenAI GPT-3.5/4 |
| 文案生成 | OpenAI GPT-3.5/4 |
| 向量嵌入 | all-mpnet-base-v2 |

## MVP 验收标准

### 功能验收

- [ ] 文本联想功能正常工作
- [ ] 联想结果合理且相关
- [ ] 热梗库可以正常浏览和搜索
- [ ] 文化库可以正常浏览
- [ ] 文案生成功能可用
- [ ] 界面美观且响应式

### 性能验收

- [ ] 页面加载 < 2s
- [ ] API 响应 < 5s
- [ ] 错误处理友好

### 质量验收

- [ ] 无严重 Bug
- [ ] 基本的错误处理
- [ ] 代码有基础注释
