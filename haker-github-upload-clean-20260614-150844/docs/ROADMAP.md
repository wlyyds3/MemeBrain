## 风险评估

| 风险 | 影响 | 概率 | 应对方案 |
|------|------|------|----------|
| AI API 不稳定 | 高 | 中 | 多服务商备用 |
| 向量检索慢 | 中 | 中 | 索引优化 + 缓存 |
| 文件上传问题 | 中 | 低 | 分片上传 + 进度 |
| 数据量增长快 | 中 | 低 | 分片 + 归档 |
